ChromaBlend

Creating vibrant color palettes, made easy.

ChromaBlend helps you create balanced color palettes. It uses a clever color system (OKLCH) to keep colors equally bright, no matter the hue. Adjust the lightness slider to see a dark palette (good with white text) and a light palette (good with black text). Every color box shows you its contrast level and its color code (hex).
