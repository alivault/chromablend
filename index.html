ChromaBlend

A color palette generator that preserves brightness levels.

With this tool, you can easily select any base color, and it will generate two palettes: a dark palette and a light palette. When the lightness slider is set in the middle, the dark palette ensures legibility with white text, while the light palette ensures legibility with black text. Each tile displays the color contrast ratio, as well as the hex code of the color.
